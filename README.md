# A lightweight design system for building Figma plugins with svelte 5

### Components:

- Button
- Icon Button
- Select Input Field
- Text Input Field

### Usage

If you want to use it in your project, run

```bash
npm install @alikimovich/figma-ui3-ds@0.0.1

```

or

```bash
bun add @alikimovich/figma-ui3-ds@0.0.1
```
